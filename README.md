# Opening Habit

Opening Habit is a free, visual chess-opening trainer. Choose a repertoire for
White or Black, select an opening and variation, then step through the moves on
an animated chessboard.

## Start the app

On a Mac:

1. Open this folder in Finder.
2. Double-click `START_APP.command`.
3. The app will open in your browser at `http://127.0.0.1:4188/`.

The launcher starts the local preview when needed. If it is already running, it
simply reuses the existing preview.

If macOS asks for confirmation the first time, allow the launcher to open.

## Features

- Visual green-and-cream chessboard with animated piece movement
- White and Black repertoire selection
- Random opening practice
- Searchable opening library
- 30 opening families
- 62 practical variations
- Plain-language explanation for each move
- Previous and next controls for reviewing a line

## Project files

- `index.html` contains the app screens.
- `styles.css` controls the layout and appearance.
- `app.js` contains the opening library and chessboard behavior.
- `assets/pieces.svg` contains the chess-piece artwork.
- `START_APP.command` launches the app locally on macOS.
